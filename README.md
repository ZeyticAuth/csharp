# ZeyticAuth C#

[![ZeyticAuth](https://img.shields.io/badge/for-zeytic-7958ff)][Website]
[![Discord](https://img.shields.io/discord/965845662535147551?color=5865f2&logo=discord&label=discord)][Discord]

The monorepo for ZeyticAuth SDKs and samples written in C#.

## Contents

### Packages

- [src/ZeyticAuth.AspNetCore.Authentication](./src/ZeyticAuth.AspNetCore.Authentication): ASP.NET Core authentication middleware for ZeyticAuth.
- [src/ZeyticAuth.AspNetCore.Authentication.Tests](./src/ZeyticAuth.AspNetCore.Authentication.Tests): Tests for the ASP.NET Core authentication middleware.

### Samples

- [sample](./samples/sample): Sample ASP.NET Core web application that shows how to use the ASP.NET Core authentication middleware.
- [sample-mvc](./samples/sample-mvc): Sample ASP.NET Core web MVC application that shows how to use the ASP.NET Core authentication middleware.
- [sample-wasm](./samples/sample-wasm): Sample Blazor WebAssembly application that shows how to use Blorc.OpenIdConnect to authenticate users with ZeyticAuth.
- [sample-blazor](./samples/sample-blazor): Sample Blazor Server application that shows how to use the ASP.NET Core authentication middleware.
