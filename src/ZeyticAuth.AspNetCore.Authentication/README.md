[![Discord](https://img.shields.io/discord/965845662535147551?color=5865f2&logo=discord&label=discord)][Discord]

ASP.NET Core authentication middleware for ZeyticAuth.

## Prerequisites

- .NET 6.0 or higher
- A [ZeyticAuth Cloud][Website] account or a self-hosted ZeyticAuth
- A ZeyticAuth traditional web application created
- An ASP.NET Core web application created

## Guide

Check out our [docs](https://docs.zeytic.com/sdk/) and choose the right guide for your project.

## Installation

```bash
dotnet add package ZeyticAuth.AspNetCore.Authentication
```
